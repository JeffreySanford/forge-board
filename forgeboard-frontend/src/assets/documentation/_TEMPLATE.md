## Example Usage

To use these templates:

1. Copy the header template at the beginning of each markdown document
2. Replace `[Document Type]` with the appropriate category (e.g., "User Guide", "Technical Reference", etc.)
3. Add the footer template at the end of each document

This ensures consistent branding and styling across all ForgeBoard documentation.
